# Repo Skills

These skills are human-first guides for repeatable product documentation work in this repo.

## Available Skills

- `sop-writing`: Use when creating or revising SOP files in `docs/products/sop/`.
- `competitor-audit-writing`: Use when creating competitor audits in `docs/products/audits/competitive/`.

## Usage

Open the relevant `SKILL.md` before starting the work. Follow the checklist closely enough that another person can understand the output without needing the original conversation.
